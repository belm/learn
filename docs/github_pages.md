* github账号 ，创建一个用来生成页面的仓库，这里假如belm/learn
* [创建token](https://github.com/settings/tokens)  用来自动提交生成后的静态页面
* [生成仓库对应的secret](https://github.com/belm/learn/settings/secrets/actions)  填写上一步生成的token，github actions自动化脚本会读取
* 准备一个域名（如果需要）
